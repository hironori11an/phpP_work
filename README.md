# BookSpace
---
# 概要   
本のレビューを投稿し、読書レビューを共有できるアプリです。
---
# URL
https://www.bookspace-review.net
---
# 機能一覧
### ホーム
- ログイン機能（一般、管理者を分けて認証）  
- ユーザ登録（一般）   
- レビュー検索
### 一般ユーザ用機能
- レビュー検索、いいね登録
- レビュー投稿
- マイレビュー編集（更新・削除）、いいね取消   
### 管理ユーザ用機能
- ユーザ登録（一般ユーザ・管理者ユーザ）
- ユーザ一覧、ユーザ別のレビュー参照
